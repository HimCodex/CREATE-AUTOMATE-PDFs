# 📊 Sales Report PDF Generator (ReportLab)

This project generates a **sales report PDF** using Python and ReportLab.

## 🚀 Features
- Creates PDF file automatically
- Adds title, table, and bar chart
- Uses real data (sales per quarter)

## 🛠️ Tech Used
- Python
- ReportLab

## 📦 Installation
```bash
pip install reportlab
```
